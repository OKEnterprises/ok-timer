# OK Timer

A minimal timer for your browser. Countdown to future dates, count up from past dates, or track time from now.

Open `index.html` and press `+` to begin.

## Stack

Vanilla HTML/CSS/JS · localStorage · Zero dependencies · No build tools

## Development

```bash
open index.html                    # macOS
start index.html                   # Windows
python -m http.server 8000         # or serve locally
```
